༄༅། །གཉལ་སྟོན་དཔལ་འབྱོར་ལྷུན་གྲུབ་ཀྱི་མཛད་རྣམ་ མདོར་བསྡུས་བཞུགས། སེ་ར་དགོན་གྱི་ཁྲི་ཐོག་བརྒྱད་པ་འདི་ཉིད་རབ་བྱུང་བདུན་པའི་མེ་ལུག་སྤྱི་ལོ་ ( ༡༤༢༧ ) ལོར་གཉལ་ཁུལ་དུ་སྐུ་བལྟམས། སྐུ་གཞོན་དུས་གསང་ཕུར་ཆོས་ཞུགས་མཛད་དེ་ཆོས་རྗེ་རིན་ཆེན་བྱང་ཆུབ་དང༌། འབྲས་སྤུངས་སུ་བྱོན་ཏེ་མཁས་པ་མུས་སྲད་པ། དགའ་ལྡན་དུ་ཕེབས་ཏེ་བ་སོ་ཆོས་ཀྱི་རྒྱལ་མཚན་སོགས་ལ་བསྟེན་ནས་མདོ་སྔགས་རིག་གནས་དང་བཅས་པར་སྦྱངས་པ་དཔྱིས་ཕྱིན་པར་མཛད་དེ་མཁས་པ་ཆེན་པོར་གྱུར། གྱེ་རེ་དཀའ་བཞི་པ་སོགས་ལ་བསྙེན་པར་རྫོགས་ཤིང་ཟབ་ཆོས་ཀྱི་དབང་ལུང་ཁྲིད་གསུམ་མང་དུ་གསན། དེ་ནས་དགའ་ལྡན་བྱང་རྩེའི་མཚན་ཉིད་གྲྭ་ཚང་གི་འཆད་ཉན་པ་མཛད། གསང་ཕུར་སྦེ་སེར་འཆད་ཉན་མཛད་སྐབས་པཎ་ཆེན་ཤཱཀྱ་མཆོག་ལྡན་དང་རྩེ་ཉག་པ་གཉིས་ཐལ་འགྱུར་གྱིས་བཅོམ་པས་མཚན་སྙན་གྲགས། གཞིས་ཀ་སྣེའུ་པའི་དཔོན་དཔལ་འབྱོར་རྒྱལ་པོ་དང༌། དཔོན་ས་བུ་ཁྲིད་དཔལ་འཛོམས་གཉིས་ཀྱིས་སེ་ར་བྱེས་གྲྭ་ཚང་བཙུགས་པའི་བྱེས་ཀྱི་སློབ་དཔོན་དང་པོ་དེ་རྗེ་འདིས་མཛད།
ལོ་བཅོ་ལྔའི་རིང་གྲྭ་ཚང་དེའི་མཚན་ཉིད་ཀྱི་འཆད་ཉན་སྤེལ། དགུང་ལོ་ལྔ་བཅུ་ང་ལྔ་པ་རབ་བྱུང་བརྒྱད་པའི་ལྕགས་མོ་གླང་ལོ་སྟེ་སྤྱི་ལོ་ ( ༡༤༨༡ ) ལོར་སེ་ར་ཐེག་ཆེན་གླིང་གི་གདན་སའི་ཁྲི་ལ་བཞུགས་ནས་ལོ་ངོ་སོ་བཞིའི་རིང་དུ་གདན་ས་བསྐྱངས་ཤིང་ཆོས་ཀྱི་འཁོར་ལོ་རྒྱ་ཆེར་བསྐོར། དགུང་ལོ་བརྒྱད་ཅུ་གྱ་བདུན་པ་རབ་བྱུང་དགུ་པའི་ཤིང་ཕོ་ཁྱི་ལོ་སྟེ་སྤྱི་ལོ་ ( ༡༥༡༤ ) ལོར་དགོངས་པ་ཆོས་དབྱིངས་སུ་གཤེགས། གསུང་འབུམ། ¡¶ རྒྱལ་བའི་གསུང་རབ་ཀྱི་དྲང་ངེས་འབྱེད་པའི་རྣམ་བཤད་ལེགས་བཤད་སྙིང་པོའི་དོན་གསལ་བར་བྱེད་པའི་བསྟན་པའི་སྒྲོན་མེ་ ¡· དང༌། ¡¶ སྙན་བརྒྱུད་ཡིད་བཞིན་ནོར་བུ་ལྟ་བ་སྤྱི་ཁྱབ་ཏུ་ངོ་སྤྲོད་པའི་ཁྲིད་ཡིག་ ¡· སོགས་ཡོད། ༈ པཎ་ཆེན་བློ་བཟང་ཆོས་ཀྱི་རྒྱལ་མཚན་གྱི་མཛད་རྣམ་ མདོར་བསྡུས་བཞུགས། ཁོང་ནི་རབ་བྱུང་བཅུ་པའི་ལྕགས་རྟ་སྤྱི་ལོ་ ( ༡༥༧༠ ) ལོར་གཙང་རོང་གི་མཚམས་ལྷན་གྱི་ནང་དྲུང་རྒྱལ་བའི་སྤེའུ་ཡུལ་དུ་ཡབ་འཚོ་བྱེད་ཚེ་རིང་དཔལ་འབྱོར་དུ་གྲགས་ཀྱང་གསོ་རིག་གི་ཡོན་ཏན་དང་ལྡན་པའི་མེས་པོ་དྲུང་འཚོ་བ་ཀུན་དགའ་འོད་ཟེར་དང༌། ཡུམ་མཚོ་རྒྱལ་གཉིས་ཀྱི་སྲས་སུ་སྐུ་འཁྲུངས། ཁོང་གི་བྱིས་པའི་མིང་ལ་ཆོས་རྒྱལ་དཔལ་ལྡན་བཟང་པོ་ཞེས་བཏགས།
དགུང་ལོ་ལྔ་ལ་ཕེབས་སྐབས་དབེན་ས་པ་བློ་བཟང་དོན་གྲུབ་ཀྱི་སློབ་མ་མཁས་གྲུབ་སངས་རྒྱས་ཡེ་ཤེས་ཀྱིས་དབེན་ས་པའི་སྤྲུལ་སྐུར་ངོས་འཛིན་ཞུས་ཏེ་མཚན་ལ་བློ་བཟང་ཆོས་ཀྱི་རྒྱལ་མཚན་ཞེས་གསོལ། དགུང་ལོ་བཅུ་བཞི་པར་དབེན་དགོན་གྱི་ཁྲི་པ་མཛད། དགུང་ལོ་བཅུ་བདུན་པར་བཀྲ་ཤིས་ལྷུན་པོར་མཚན་ཉིད་སློབ་གཉེར་ལ་ཕེབས། དགུང་ལོ་ཉེར་གཉིས་པར་བཀྲ་ཤིས་ལྷུན་པོའི་ཁྲི་པ་དམ་ཆོས་ཡར་འཕེལ་ལས་བསྙེན་རྫོགས་ཀྱི་སྡོམ་པ་བཞེས། དགུང་ལོ་ཉེར་དགུ་པར་ལྔ་མཆོད་ཆེན་མོའི་ཉིན་གངས་ཅན་ཆོས་འཕེལ་གྱི་ཁྲིར་ཕེབས། དགུང་ལོ་སོ་གཅིག་པར་བཀྲ་ཤིས་ལྷུན་པོའི་ཁྲི་ཐོག་ཏུ་ཕེབས། དགུང་ལོ་ཞེ་བཞི་པར་རྒྱལ་བ་ཡོན་ཏན་རྒྱ་མཚོའི་ཡོངས་འཛིན་མཛད། དགུང་ལོ་ཞེ་བདུན་ནས་བཟུང་སེ་འབྲས་གཉིས་ཀྱི་མཁན་པོ་ལོ་ལྔ་རེ་མཛད། དགུང་ལོ་ལྔ་བཅུ་པར་ལྷ་སའི་ལྕགས་པོ་རིའི་མཚམས་སུ་བོད་སོག་དམག་འཁྲུག་ལངས་སྐབས་ཆིངས་ཞུ་ནན་ཆེན་མཛད་པས་དམག་མཚམས་བཞག ། དགུང་ལོ་ལྔ་བཅུ་ང་གཉིས་པར་རྒྱལ་བ་བློ་བཟང་རྒྱ་མཚོ་རྒྱལ་བའི་ཡང་སྲིད་དུ་ངོས་འཛིན་མཛད།
དགུང་ལོ་ང་ལྔ་བར་རྒྱལ་དབང་ལྔ་པའི་རབ་བྱུང་གི་མཁན་པོ་མཛད། དགུང་ལོ་རེ་བརྒྱད་པར་རྒྱལ་དབང་ལྔ་པར་བསྙེན་རྫོགས་ཀྱི་སྡོམ་པ་ཕུལ། དགུང་ལོ་དོན་གཉིས་ནས་བཟུང་ཞ་ལུ་དགོན་གྱི་མཁན་པོ་ལོ་ཉི་ཤུར་མཛད། དགུང་ལོ་དོན་བཞི་པར་སེ་ཆེན་ཆོས་རྗེ་རྒྱ་ནག་ཏུ་ཆེད་གཏོང་མཛད་པས། གོང་མ་ཤུན་ཊིས་ ( བླ་ན་མེད་པ་ ) ཞེས་པའི་མཚན་གསོལ་བར་མ་ཟད་གསེར་ཡིག་པ་ཆེད་གཏོང་གིས་ཐམ་ག་སྐྱེས་དང་བཅས་པ་གནང༌། ( རང་རྣམ་ནང་དུ་གསལ། ) དགུང་ལོ་དོན་ལྔ་བར་གུ་ཤྲཱི་བསྟན་འཛིན་ཆོས་རྒྱལ་གྱིས་པཎ་ཆེན་བྷོ་ཁི་ཏོ་ཞེས་པ་སྟེ་བློ་གྲོས་སྟོབས་ལྡན་ཞེས་པའི་མཚན་སྙན་གསོལ། དེ་ནས་བཟུང་པཎ་ཆེན་སྐུ་ཕྲེང་རྣམས་ཀྱི་མཚན་དུ་པཎ་ཆེན་ཞེས་ཆེད་དུ་ཐོགས་པ་རེད། ཁོང་ནི་པཎ་ཆེན་སྐུ་ཕྲེང་བཞི་པ་ཡིན། རྒྱལ་དབང་ལྔ་པས་ཁོང་གི་སྤྲུལ་སྐུ་ངོས་འཛིན་གནང་བ་མ་ཟད།

དེ་ནས་ལྕང་སྐྱ་གོང་མའི་སློབ་མ་ཨ་རྩེ་ཆོས་རྗེ་བློ་བཟང་ཆོས་འཛིན་སློབ་དཔོན་དུ་བསྟེན་ནས་ཡི་གེའི་འབྲི་ཀློག་དང་ཆོས་སྤྱོད་གསོལ་འདེབས་བཅས་ལ་ལེགས་པར་སྦྱངས། དགུང་ལོ་བདུན་པར་རྗེ་བློ་བཟང་བསྟན་འཛིན་རྒྱལ་མཚན་གྱི་དྲུང་ནས་དགེ་བསྙེན་གྱི་སྡོམ་པ་བཞེས། དགུང་ལོ་བརྒྱད་པར་གོང་མ་ཡུང་ཊིང་གིས་གདན་འདྲེན་ཞུས་པ་ལྟར་པེ་ཅིན་དུ་ཕེབས་ནས་ཟུང་གྲུ་ཟི་དགོན་པའི་ཁྲི་ལ་ཞབས་ཀྱི་པདྨོ་བཀོད། གོང་མས་ལྕང་སྐྱ་ལ་བྱམས་བརྩེས་འགྲོ་བ་ཡོངས་ཀྱི་སྤྱི་བོ་ནས་དབང་བསྐུར་བའི་ཀུན་མཁྱེན་བླ་མ་ཞེས་པའི་མཚན་དང༌། གསེར་སྲང་བརྒྱད་ཅུ་ལས་བྱས་པའི་ཐམ་ག་གསེར་ཡིག་བཅས་གནང་སྟེ་ཆབ་སྲིད་ཆེན་པོའི་བླ་མར་བཀུར། དགུང་ལོ་བཅུ་དགུ་པར་གོང་མའི་བཀའ་ལྟར་དབུས་སུ་ཕེབས་ཏེ་རྒྱལ་དབང་སྐལ་བཟང་རྒྱ་མཚོ་ཁྲི་ལ་བཀོད་པར་མཛད། པཎ་ཆེན་བློ་བཟང་ཡེ་ཤེས་ཀྱི་དྲུང་ནས་དགེ་ཚུལ་དང་དགེ་སློང་གི་སྡོམ་པ་མནོས་ཏེ་མཚན་ལ་ཡེ་ཤེས་བསྟན་པའི་སྒྲོན་མེ་ཞེས་གསོལ། དགུང་ལོ་ཉེར་ལྔ་པར་གོང་མ་ཆན་ལུང་གི་བཀའ་ལྟར་བསྟན་འགྱུར་སོག་ཡིག་ཏུ་བསྒྱུར་པས་ཕྱིས་སུ་པེ་ཅིང་དུ་པར་བརྐོས་ཡོད། དེ་ནས་མི་རིང་བར་དགའ་ལྡན་བྱིན་ཆགས་གླིང་དང་དབེན་དགོན་དམ་ཆོས་དགའ་ཚལ། གདུགས་དཀར་ལྷ་ཁང་སུམ་ཐོག་ཅན།
རྒྱ་གར་ལྷ་ཁང༌། ཞེ་ཧོར་སྐྱེད་ཚལ་གྱི་ཉེ་འདབས་སུ་བསམ་ཡས་གཙུག་ལག་ཁང་ལ་དཔེ་བླངས་ནས་བཞེངས་པའི་དགོན་པ་སོགས་པེ་ཅིན་དུ་བོད་ལུགས་ཀྱི་དགོན་པ་མང་པོ་གསར་དུ་བཞེངས་ཏེ་གོང་མ་ལའང་བོད་ཡིག་བསླབས་ཤིང་ཟབ་ཆོས་རྣམས་ཀྱང་རིམ་གྱིས་ཕུལ། དགུང་ལོ་ཞེ་གཅིག་པར་གོང་མའི་བཀའ་བཞིན་རྒྱལ་བ་རིན་པོ་ཆེའི་ཡང་སྲིད་བཙལ་བར་དབུས་སུ་ཕེབས། དགུང་ལོ་ང་ལྔ་པར་གོང་མའི་བཀའ་བཞིན་བཀའ་འགྱུར་མན་ཇུའི་སྐད་ཡིག་ཏུ་བསྒྱུར་པས་གོང་མ་ཐུགས་ཤིན་ཏུ་དགྱེས་ནས་ཁོང་ལ་བརྩི་བཀུར་མཛད་དོ། ། རབ་བྱུང་བཅུ་གསུམ་པའི་མེ་ཕོ་རྟ་སྟེ་སྤྱི་ལོ། ( ༡༧༨༦ ) ལོར་རི་བོ་རྩེ་ལྔར་དགོངས་པ་ཆོས་དབྱིངས་སུ་གཤེགས། ཁོང་གི་གསུང་རྩོམ་ནི་བསྟན་བཅོས་ཆེ་ཆུང་ཆིག་བརྒྱ་བརྒྱད་ཅུ་གྱ་དགུ་ཡོད་པ་རྣམས་པོ་ཏི་བདུན་དུ་བསྡུས་ཏེ་པེ་ཅིན་དུ་ཤིང་པར་བརྐོས་ཡོད། ཁོང་གི་སློབ་མ་ནི་དམ་པའི་སྐྱེས་རབས་འཛིན་པའི་སྤྲུལ་སྐུ། ཆོས་ཀྱི་གདན་ཁྲིར་བཞུགས་པའི་བླ་ཆེན། བཀའ་པོད་མང་པོ་སྦྱངས་པའི་དགེ་བཤེས།
སྒྲུབ་པ་སྙིང་པོར་བྱེད་པའི་སྤོང་བ་པ། སྐད་གཉིས་སྨྲ་བའི་ལོ་ཙཱ་བ། འདུལ་བ་དང་མདོ་སྡེ་ལ་མཁས་པའི་ཧྭ་ཤང༌། དད་པའི་ནོར་གྱི་ཕྱུག་པའི་ཁྱིམ་བདག་སོགས་མཐའ་དབུས་ནོར་འཛིན་གྱི་ཁྱོན་གང་སར་གནས། ༈ ལྟ་བའི་ཉམས་མགུར་ཨ་མ་ངོས་འཛིན་གྱི་འགྲེལ་པ་བདག་འཛིན་དགྲ་དཔུང་འཇོམས་པའི་མཚོན་ཆ་སྐལ་ལྡན་པདྨོ་བཞད་པའི་ཉིན་བྱེད་ཅེས་བྱ་བའི་རྩོམ་པ་པོ་གསལ་བར་མ་ངེས་པས་མཛད་རྣམ་འབྲི་བར་མ་སྤོབས། འོན་ཀྱང་འདིའི་མཇུག་ཏུ་མཁས་གྲུབ་ཀུན་གྱི་གཙུག་རྒྱན་ལྕང་སྐྱ་རོལ་པའི་རྡོ་རྗེའི་ཉམས་མགུར་འདི་ལ་སྔགས་དང་སྦྱར་བའི་འགྲེལ་པ་ཞིག་དགོས་ཞེས་མཚུངས་བྲལ་དམ་པའི་སྐྱེས་མཆོག་ལྕང་སྐྱ་མཆོག་སྤྲུལ་རིན་པོ་ཆེ་དང༌། སྒྲུབ་ཁང་སྤྲུལ་པའི་སྐུ་བཅས་ནས་བསྐུལ་ངོར་དམ་པ་གང་དེ་ལ་དད་པ་དང་ལྡན་པ་༧གོང་མ་ཆེན་པོའི་ལུང་གིས་ཞྭ་སེར་བསྟན་པའི་གསལ་བྱེད་ཨ་ཆི་ཐུ་ནོ་མན་ཧན་དུ་འབོད་པས་བཞི་སྡེ་དགའ་ལྡན་བསམ་གཏན་གླིང་གི་རང་ཤག་ཟུང་འཇུག་བདེ་བ་ཆེན་པོའི་ཕོ་བྲང་དུ་སྦྱར་བའི་ཡི་གེ་པ་ནི་དད་བརྩོན་དགེ་ལ་འདུན་པ་དྲུང་ཡིག་དགེ་སློང་རྒྱ་མཚོ་དཔལ་འབར་གྱིས་བགྱིས་པ་ཛ་ཡནྟུ་ཞེས་གསལ་བར་དཔག་ན་དུས་རབས་བཅུ་དགུ་པའི་ནང་དུ་བྱུང་བའི་རི་བོ་དགེ་ལྡན་པའི་མཁས་གྲུབ་ཞིག་གོ ། རྩོམ་པ་པོའི་རྣམ་ཐར་མདོར་བསྡུས། རྩོམ་པོའི་རྣམ་ཐར་མདོར་བསྡུས། - PAGE 6 -- PAGE 7 -- PAGE 1 -
